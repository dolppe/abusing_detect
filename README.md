# abusing_detect
배틀로얄 게임(이터널 리턴)에서의 티밍 행위 분석 및 식별.


## Summary
이터널 리턴 게임에는 티밍이라는 어뷰징 행위가 존재합니다.  
티밍은 팀을 맺는 것이 허용되지 않은 게임에서 다른 사람과 팀을 맺어 싸우는 행위입니다.  
티밍 행위를 식별하기 위해 여러 가정과 방안을 고려하여 분석을 진행했습니다.  

핵심 가정은 티밍 유저는 모든 판마다 티밍을 하는 것이 아니고, 티밍 유무에 따른 실력 차이가 존재한다는 점입니다.  
클러스터링을 이용하여 티밍에 대한 이상치를 탐지하고, 데이터를 분석하였습니다.

결론이 담긴 문서입니다.  
(https://drive.google.com/file/d/1b543mzdKKOo1XR455I0V5DVR5DMN4fFz/view?usp=drive_link)

## Usage
클러스터링은 main_clustering.ipynb 파일을 이용하여 진행하면됩니다.  
이에 필요한 주데이터의 경우, 용량이 커서 업로드하지 않았습니다.  
이는 이터널리턴 API를 이용하여 따로 데이터를 구축한 후에 진행하거나 코드에서 직접 요청하여 구축할 수 있습니다.


## Requirements
- Python 3.9.7
- 이터널리턴 API (https://developer.eternalreturn.io/)
